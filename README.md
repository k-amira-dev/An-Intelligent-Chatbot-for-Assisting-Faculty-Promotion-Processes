# An-Intelligent-Chatbot-for-Assisting-Faculty-Promotion-Processes
This project is a web-based academic chatbot designed to support faculty promotion–related inquiries at the University of Sharjah. The system uses Retrieval-Augmented Generation (RAG) to provide accurate, policy-grounded responses based on official university documentation, specifically the Faculty Handbook.

Unlike traditional chatbots or standalone large language models, this system retrieves relevant sections from authoritative documents and uses them to generate informed answers, reducing the risk of misinformation and hallucination. The chatbot is intended as an advisory tool, helping faculty members better understand promotion requirements while improving clarity and consistency in information access.

The project is developed as part of a junior-level computer science course, focusing on core RAG concepts, backend integration, and a simple web-based user interface. Advanced features such as agentic AI behavior and personalized workflows are intentionally excluded and reserved for future work.
